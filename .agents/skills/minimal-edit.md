# Minimal Edit

Edite apenas o necessário para cumprir o solicitado.

## Regras estritas:

- Altere somente o arquivo indicado
- Faça somente a mudança pedida, nada mais
- Sem fallbacks, sem guards, sem aliases, sem abstrações extras
- Sem linter fixes, sem type fixes, sem refatorações colaterais
- Sem comentários explicativos adicionados ao código
- Se a mudança for uma linha, edite uma linha
